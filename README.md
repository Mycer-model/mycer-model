# MYCER Replication Package

This repository contains the replication package for the MYCER
(Mustafa Yüksel Çınar Energy Resilience Index) framework developed
in the doctoral dissertation.

The repository provides the computational workflow used in the thesis:

1. Data preprocessing
2. Indicator normalization
3. Data Fidelity Correction (DFC)
4. Hybrid weighting (AHP + Entropy + SHAP)
5. Regional Scaling Factor (RSF)
6. Structural vulnerability calculation
7. MYCER composite index computation
8. Sensitivity analysis

## Repository Structure

data/ – raw and processed datasets  
scripts/ – Python scripts implementing MYCER calculations  
notebooks/ – Jupyter notebooks for reproducibility  
docs/ – methodological documentation  
figures/ – generated visualizations

## Computational Environment

Python libraries used:

- pandas

- scikit-learn
- xgboost
- shap

## Replication Package

This repository provides the computational replication package for the MYCER framework presented in the doctoral dissertation.

All scripts necessary to reproduce the MYCER index calculation, including data preprocessing, weighting procedures, structural vulnerability computation, and final aggregation, are documented within this repository.

The repository is intended to support transparency, reproducibility, and open scientific practice.
